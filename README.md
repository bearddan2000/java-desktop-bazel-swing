# java-desktop-bazel-swing

## Description
This is a POC project to demonstrate a
java swing application running inside a
docker container.

## Tech stack
- bazel
- java
  - swing

## Docker stack
- l.gcr.io/google/bazel:latest
- alpine:edge

## To run
`sudo ./install.sh -u`

## To see help
`sudo ./install.sh -h`
